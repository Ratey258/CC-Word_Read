# 文件导入功能 - 使用指南

## 🎉 新功能

现在您可以通过"文件"菜单的"打开文件"功能导入多种格式的文档，并完美保留原文档格式！

## 📋 支持的文件格式

- ✅ **TXT** - 纯文本文件
- ✅ **DOCX** - Microsoft Word 文档（完美保留格式）
- ✅ **MD** - Markdown 文件

## 🚀 如何使用

### 方式 1：通过文件菜单
1. 点击左上角"文件"选项卡
2. 点击"打开文件"（或按 `Ctrl+O`）
3. 选择您想要导入的文件
4. 文件会自动导入并保留原有格式

### 方式 2：快捷键
- 按 `Ctrl+O` 直接打开文件选择对话框

## ✨ 格式保留说明

### Word 文档 (.docx)
完美保留以下格式：
- 📝 标题样式（H1, H2, H3等）
- **粗体**、*斜体*、<u>下划线</u>
- 有序列表和无序列表
- 段落结构和缩进
- 其他 Word 内置格式

### Markdown 文件 (.md)
自动转换以下语法：
- `# 标题` → 标题样式
- `**粗体**` → **粗体**
- `*斜体*` → *斜体*
- 列表项 → 列表格式

### 纯文本 (.txt)
- 保持原有的段落结构
- 转换为标准的文档格式

## 💡 提示

- 导入成功后会显示提示信息
- 如果文档包含格式，会提示"（已保留格式）"
- 大文件可能需要几秒钟来解析

## 📦 技术栈

- **mammoth.js** - Word 文档解析
- **自定义 Markdown 解析器** - Markdown 格式支持
- **TypeScript** - 类型安全的实现

---

**注意**: 原来的"📖导入"按钮已被移除，所有导入功能统一在"文件"→"打开文件"中。
